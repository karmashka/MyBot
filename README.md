# MyBot
Для базовой части надо: науситься делать запросы в википедию по слову и стягивать первые несолько строк.

Сделано: бот @MariK_bot умеет ходить в википедию, искать статью по слову, брать несколько первыx строк статьи. Причем поисковое слово должно соотвествовать уникальному названию статьи (т.е. если в википедии есть несколько статей с таким названием или не совсем четко сформулирован запрос, и есть несколько похожих на поисковое слово статей- бот пишет, что ему непонятен запрос). 
Также бот умеет выдавать последние 10 запросов (историю).

Бот развернут на Heroku. Блокировка преодолена путем установки Windscribe.
